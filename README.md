# 三省六部回测内阁

这是一个基于“三省六部”架构的高频股票回测系统，旨在对10套独立交易策略进行全自动回测。

## 架构说明

### 三省
- **太子 (Crown Prince)**: 数据前置校验与分发。
- **中书省 (Zhongshu Sheng)**: 策略信号生成。
- **门下省 (Menxia Sheng)**: 风控审核（一票否决）。
- **尚书省 (Shangshu Sheng)**: 执行与资金管理。

### 六部
- **吏部 (Personnel)**: 策略管理。
- **户部 (Revenue)**: 资金核算。
- **礼部 (Rites)**: 业绩报表。
- **兵部 (War)**: 模拟成交。
- **刑部 (Justice)**: 合规记录。
- **工部 (Works)**: 数据工程。

## 运行方式

```bash
python main.py
```
